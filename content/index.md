**Cassiopaean Cosmology** is a project of the [Cassiopaea.org](https://cassiopaea.org/)community, aiming to provide an organized overview of the different events from <a href="https://cassiopaea.org/forum/boards/cassiopaean-session-transcripts.51/" target="_blank" rel="noopener noreferrer"> cassiopaean sessions</a>.

This is created based on the cassiopean experiment framework.
